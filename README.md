# Reportes Automatizados API

Este es el backend para la aplicación de generación de reportes automatizados.

## Descripción

La API está construida con Django y Django REST Framework, y proporciona endpoints para gestionar un CRUD completo, así como funcionalidades para la carga y procesamiento de archivos de Excel.

## Entorno de Desarrollo

El proyecto está dockerizado para facilitar el desarrollo. Para levantar el entorno, asegúrate de tener Docker y Docker Compose instalados y ejecuta:

```bash
docker-compose up --build
```
